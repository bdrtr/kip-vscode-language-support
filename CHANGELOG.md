# Changelog

All notable changes to the Kip Language Extension will be documented in this file.

## [1.1.0] - 2026-01-23

### Added
- ✅ Complete LSP integration with all providers
- ✅ Go to Definition (F12)
- ✅ Find References (Shift+F12)
- ✅ Rename Symbol (F2)
- ✅ Code Actions (Ctrl+.)
- ✅ Code Lens (reference count)
- ✅ Document Symbols (Outline)
- ✅ Workspace Symbols (Ctrl+T)
- ✅ Automatic dependency checking scripts
- ✅ VSIX validation scripts
- ✅ Comprehensive error handling and fallback mechanisms

### Fixed
- ✅ Fixed missing `semver` and `balanced-match` dependencies
- ✅ Fixed VSIX packaging to include all required modules
- ✅ Improved error handling for semantic tokens
- ✅ Reduced log spam for unsupported LSP features
- ✅ Fixed terminal creation fallback to Output Channel

### Improved
- ✅ Better error messages and logging
- ✅ More robust LSP initialization
- ✅ Improved fallback mechanisms when LSP features are unavailable
- ✅ Better TypeScript type safety

## [1.0.9] - 2025-01-XX

### Added
- Basic language support
- Syntax highlighting
- Code completion
- Hover documentation
- Code formatting
- Run command
